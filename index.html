shapeshifter.js

a simple svg shape maker

Shapeshifter.js allows you to easily convert divs into simple and fun SVG shapes!

info
Download
docs

Customize your shape simply by passing a few options into the shapeshifter function:

		
	 $('#div').shapeshifter({
	    	shape: "hexagon",
	    	fill: "#303F9F",
	    	fillImage: "images/image.gif",
	    	multiplier: 2,
	    	rotation: -30,
	    	rotationOffset: "true"
	    });

multiplier: 2

Standard shapes are 100px x 100px. Size them by changing the multiplier value.

fills

solid colour

static image

animated .gif

fill: "#FF5722"

Input a color value to fill with a solid colour.

fillImage: "images/image.gif"

Input the image path relative to the div to change the fill of the shape to an image - even an animated .gif! Note that fillImage will override a fill colour, so simply don't include it if you'd like to use a plain colour.

rotation

rotation only

rotationOffset

180 degrees + rotationOffset

rotation: -30

Input a value in degrees to rotate the shape

rotationOffset: "true"

Set to a string of "true" to set the contents of the div back to plain ol' horizontal.
